## 二、学习步骤

根据算法分类进行刷题，基本的、变异的

解题步骤：

1、看懂题目

2、分析、推导解法

3、思路转变成代码

4、牢记模板（根据题目，能立马想出几种对应的解法）

5、持续的学习和做题

**最重要的，要有产出**
